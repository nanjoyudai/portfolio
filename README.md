- ### 6. テーブルデータに対する各種機械学習モデルの実装と性能比較 (ワイン品種分類)
- **What & Why (目的):** 明示的な特徴量（アルコール度数、色合いなど13種類）を持つワインのデータセットを用い、3つの品種に分類する予測モデルを構築しました。性質の異なる複数の機械学習アルゴリズムを実装・比較し、それぞれの特性と予測精度を評価することを目的としています。
- **How (使用技術):** Python, scikit-learn, Jupyter Notebook (決定木, ロジスティック回帰, 多層パーセプトロン, ランダムフォレスト)
- **工夫した点・困難だった点:** 単にモデルを適用するだけでなく、未知のデータに対する汎化性能を高める（過学習を防ぐ）ためのアプローチを徹底しました。
  1. ロジスティック回帰では事前にデータの標準化を行い、正則化パラメータを交差検証（クロスバリデーション）で最適化しました。
  2. 多層パーセプトロン（MLP）では、隠れ層の層数・次元数（50〜150）を複数パターン定義し、交差検証により最適なネットワーク構造（1層・100ノード）を探索しました。
  3. モデルの「ブラックボックス化」を防ぐため、決定木の分岐条件の可視化や、ロジスティック回帰・ランダムフォレストにおける特徴量の寄与度（係数）をグラフ化し、どの変数が分類に強く寄与しているかを論理的に考察しました。
- **結果:** 各モデルで高い精度を出し、特にランダムフォレストにおいてはテストデータに対して正解率1.000（100%）という極めて高い汎化性能を達成しました。
